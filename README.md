# weather-app-
A weather monitoring app with as many features as i can add
A simple Python-based weather app that fetches real-time weather data using the OpenWeatherMap API.

Features(for now)
- Fetches live weather data
- Displays temperature and description
- Easy-to-use command-line interface

How to Run
- Clone the repo
- Install dependencies (requests)
- Add your OpenWeatherMap API key
- Run the script and enter a city name
